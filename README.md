# 🔥 坦克大战 - Tank Battle Game

一个使用 HTML5 Canvas 构建的经典坦克大战游戏，带有音效、时间奖励系统和钢铁掩体！

## 🎮 在线试玩

**👉 [点击这里开始游戏](https://forrest923.github.io/tank-battle-game/)**

## 🎯 游戏特性

### 🎵 音效系统
- ✅ 射击音效
- ✅ 爆炸音效
- ✅ 击中钢墙音效
- ✅ 游戏胜利/失败音效

### 🎮 游戏玩法
- ✅ 玩家坦克（绿色）- WASD 控制移动
- ✅ 敌方坦克AI（红色）- 自动追踪和射击
- ✅ 砖墙障碍物（棕色）- 可被炮弹摧毁
- ✅ 钢铁掩体（银色）- **不可摧毁**
- ✅ 子弹碰撞检测
- ✅ 生命系统（3条命）
- ✅ 计分系统
- ✅ 爆炸粒子效果
- ✅ 游戏胜利/失败判定

### ⏱️ 时间奖励系统
- 60秒内通关：奖励 **500分**
- 每超过10秒：减少 **50分**
- 鼓励快速通关！

## 🕹️ 操作说明

| 按键 | 功能 |
|------|------|
| W | 向上移动 |
| A | 向左移动 |
| S | 向下移动 |
| D | 向右移动 |
| 空格键 | 发射子弹 |

## 🏆 游戏目标

消灭所有敌方坦克即可获得胜利！小心躲避敌人的子弹，利用钢铁掩体保护自己。

## 🛠️ 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES6+)
- Web Audio API（音效生成）
- 无外部依赖

## 🚀 本地运行

直接在浏览器中打开 `index.html` 文件即可运行游戏。

或者使用本地服务器：

```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .
```

然后访问 `http://localhost:8000`

## 🔊 音效说明

游戏使用 Web Audio API 实时生成音效，无需外部音频文件。如果浏览器阻止了自动播放，按任意键即可启用音效。
