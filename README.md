# 🔥 坦克大战 - Tank Battle Game

一个使用 HTML5 Canvas 构建的经典坦克大战游戏。

## 🎮 在线试玩

**👉 [点击这里开始游戏](https://forrest923.github.io/tank-battle-game/)**

## 🎯 游戏特性

- ✅ 玩家坦克（绿色）- WASD 控制移动
- ✅ 敌方坦克AI（红色）- 自动追踪和射击
- ✅ 砖墙障碍物系统
- ✅ 子弹碰撞检测
- ✅ 生命系统（3条命）
- ✅ 计分系统
- ✅ 爆炸粒子效果
- ✅ 游戏胜利/失败判定

## 🕹️ 操作说明

| 按键 | 功能 |
|------|------|
| W | 向上移动 |
| A | 向左移动 |
| S | 向下移动 |
| D | 向右移动 |
| 空格键 | 发射子弹 |

## 🏆 游戏目标

消灭所有敌方坦克即可获得胜利！小心躲避敌人的子弹，保护好自己的坦克。

## 🛠️ 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES6+)
- 无外部依赖

## 📁 文件结构

```
tank-battle-game/
├── index.html    # 游戏主页面
├── game.js       # 游戏逻辑
└── README.md     # 项目说明
```

## 🚀 本地运行

直接在浏览器中打开 `index.html` 文件即可运行游戏。

或者使用本地服务器：

```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .
```

然后访问 `http://localhost:8000`
